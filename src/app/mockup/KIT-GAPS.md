# Kit-member gaps found by the design mockups

Composed page-locally in the mockups under this directory; each is flagged for
real kit work in `packages/ui` (intent block, every state, showcase specimen)
before the surfaces it serves are built. Nothing here was added to the kit in
the mockup session.

| Gap                                           | Found in                                                                            | Suggested shape                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| --------------------------------------------- | ----------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Tabs / segmented control**                  | `team/` (`TabBar`)                                                                  | Sticker-faced tab pair with `role="tablist"`, selected = pressed face + tint. The games-first plan already names this a kit addition, not a page invention.                                                                                                                                                                                                                                                                                               |
| **Numeric stepper**                           | `create-game/` (`Stepper`)                                                          | `− value +` from icon buttons with min/max, disabled faces, and an error/helper slot. StepSlider fits 4 stops, not 2–15.                                                                                                                                                                                                                                                                                                                                  |
| **Textarea**                                  | `admin-questions/` editor                                                           | TextField is the kit's only free-text entry and is single-line; OpenTDB prompts run long. Multi-line paper-sticker field.                                                                                                                                                                                                                                                                                                                                 |
| **Status line (banners' function)**           | `team/` lock lines, `admin-questions/` load error                                   | UI.md's banners law has no member: one quiet docked line — status hue + dot + text + optional inline action. Hand-assembled three times here.                                                                                                                                                                                                                                                                                                             |
| **Skeleton rows**                             | `admin-questions/` list loading                                                     | Product register wants skeletons over spinners; composed from `bg-s4` bars + `animate-pulse` + `motion-reduce:animate-none`.                                                                                                                                                                                                                                                                                                                              |
| **Pagination footer**                         | `admin-questions/`                                                                  | Range text (`1–25 of 412`) + prev/next with bound-disabled faces. Could be a documented pattern instead of a member.                                                                                                                                                                                                                                                                                                                                      |
| **Selectable sticker card**                   | `create-game/` pool picker                                                          | The selection-marker law (tint + trailing check) exists only in MenuItem; a card-scale toggle (`aria-pressed`) is needed by the pool picker and any future multi-select.                                                                                                                                                                                                                                                                                  |
| **Rejection-shake export**                    | every mockup + `showcase/motion-demos`                                              | The shake keyframes `[0,−9,8,−6,5,−3,0]` are copied in five places. Export a `SLIP_SHAKE` constant (same crossing-by-constant principle as `SLIP_EASE`).                                                                                                                                                                                                                                                                                                  |
| **Score-pop moment export**                   | `trivia/` (`ScorePop`), showcase demo                                               | The springy hand-font `+N` pop is likewise copied choreography; export or member.                                                                                                                                                                                                                                                                                                                                                                         |
| **Form field wrapper**                        | `admin-questions/` (`Field`), `create-game/` (`FieldLabel`), M4 harness (`Labeled`) | Caps label + control + error/helper line; three local variants already exist.                                                                                                                                                                                                                                                                                                                                                                             |
| **CopyCode size**                             | `create-game/` landing                                                              | The huge landing code is `className="text-4xl"` from outside; consider a sanctioned size variant.                                                                                                                                                                                                                                                                                                                                                         |
| **Form-sized Select (width/height/disabled)** | `admin-questions/` difficulty filter                                                | The kit `Select` is a compact accent chip that resizes to its value and exposes no `className`, width, height, or `disabled` control, so it can't hold a constant footprint beside an inline field. The mockup builds the dropdown page-locally from `PopoverCard` + `MenuItem` with a fixed-width, field-height sticker trigger. The kit wants a form-register `Select` variant: caller-set width, height that matches `TextField`, and a disabled face. |

Non-kit findings for the builders:

- `lobby-view.tsx` marks the leader with the word "leader" — the markers-are-
  icons law wants an icon (the mockups use ★ with an accessible name).
- The home join form reports a bad code without the rejection shake; UI.md
  names the shake as the form-error affordance (the `home/` mockup shows it).
- The trivia `Rope` is a game-surface indicator, not a kit member (game
  surfaces design their own indicators against the status vocabulary per
  UI.md); it should ship with the trivia surface, intent-documented there.
